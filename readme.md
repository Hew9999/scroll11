# BlackBox: scroll

### Requirements:

- python 3.10+

### Install:

``` 
python3 -m venv venv
source venv/bin/activate
pip3 install wheel
pip3 install -r requirements.txt
```

#### Add config files:

``` 
cp config_sample config -R
```

Add required configs in   config/* files.

### Usage:

``` 
source venv/bin/activate
python3 main.py
```

## General NOTES

All Ranges support:

- single value (example: 1)
- range (example: 1-5)
- empty value, for all items in the list
